ParseLiveListWidget

ParseLiveListWidget<T extends ParseObject> class

A widget that displays a live list of Parse objects.

The ParseLiveListWidget is initialized with a query that retrieves the objects to display in the list. The childBuilder function is used to specify how each object in the list should be displayed.

The ParseLiveListWidget also provides support for error handling and lazy loading of objects in the list.
